Mis Habilidades Técnicas

Explora en detalle las tecnologías y herramientas con las que construyo soluciones innovadoras.

Frontend

Avanzado

Java

Lenguaje de programación robusto, orientado a objetos y ampliamente utilizado en el desarrollo empresarial.

Base de mi desarrollo backend, utilizado para construir APIs, lógica de negocio y microservicios.

CSS3

Lenguaje de hojas de estilo para diseñar y dar apariencia visual a las páginas web.

Experiencia en selectores avanzados, Flexbox, Grid, animaciones y diseño responsivo. He trabajado con preprocesadores como Sass.

JavaScript (ES6+)

Lenguaje de programación esencial para la interactividad y dinamismo en el frontend.

Manejo de DOM, eventos, asincronía (promesas, async/await) y frameworks modernos.

React

Biblioteca de JavaScript para construir interfaces de usuario interactivas y reutilizables basadas en componentes.

Desarrollo de SPAs (Single Page Applications), manejo de estado (Context API, Redux básico) y hooks.

Angular

Framework de desarrollo frontend mantenido por Google para crear aplicaciones web robustas y escalables.

Conocimiento de componentes, servicios, routing y directivas, aplicado en proyectos académicos.

Backend

Java

Lenguaje de programación robusto, orientado a objetos y ampliamente utilizado en el desarrollo empresarial.

Base de mi desarrollo backend, utilizado para construir APIs, lógica de negocio y microservicios.

Spring Boot

Framework que simplifica la creación de aplicaciones Java autónomas y listas para producción basadas en Spring.

Extensamente utilizado para desarrollar APIs RESTful, microservicios, y gestionar la seguridad y persistencia de datos.

Node.js

Entorno de ejecución de JavaScript del lado del servidor, ideal para aplicaciones rápidas y escalables.

Utilizado para construir APIs y servicios backend, especialmente en conjunto con Express.js.

Express.js

Framework minimalista y flexible para Node.js, utilizado para construir aplicaciones web y APIs.

Implementación de rutas, middleware y manejo de peticiones HTTP en proyectos con Node.js.

Bases de Datos

MySQL

Sistema de gestión de bases de datos relacional de código abierto, popular y robusto.

Diseño de esquemas, consultas SQL complejas, y gestión de datos para aplicaciones web y móviles.

MongoDB

Base de datos NoSQL orientada a documentos, flexible y escalable para aplicaciones modernas.

Modelado de datos no relacionales, consultas y agregaciones para proyectos con requisitos de flexibilidad.

Herramientas y Otros

Git & GitHub

Sistema de control de versiones distribuido y plataforma de alojamiento para desarrollo colaborativo.

Uso diario para gestión de código, ramificación, fusiones y colaboración en proyectos.

SAP CPI / PI/PO

Plataformas de integración de SAP para conectar sistemas SAP y no-SAP, on-premise y en la nube.

Plataforma de colaboración para el desarrollo de APIs, utilizada para diseñar, construir, probar y documentar APIs.

Herramienta esencial para probar y depurar APIs REST y SOAP durante el desarrollo.
